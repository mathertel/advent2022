About Advent Calendar 2022

This Advent Calendar is hosted in https://mathertel.github.io/advent2021/index.htm and source code available on https://github.com/mathertel/advent2021.

An Advent calendar is a special calendar with 24 secrets behind doors used to count the days of Advent in anticipation of Christmas. You can find also many online following the tradition founded by the German Lutherans in the 19th and 20th centuries.

This one was created from my personal collection of useful links and snippets regarding the possibilities of the native browser technologies in 2021/2022.

It focuses on using the CSS, JavaScript and HTML broadly available and usable as of 2021 like CSS3, Promises, WebControls and other things I discovered and checked in 2021. New features may not actually come from 2021 but they get important and useful.

In adition some articles are on using javascript (and typescript) on a nodejs based server.

The doors will be available on each of the first 24 days in December 2021 and will stay open.

Client Source Code is plain HTML, JavaScript and CSS - no frameworks.

Server Source Code is plain JavaScript with few libraries included.

Happy reading. Feel free to share.
